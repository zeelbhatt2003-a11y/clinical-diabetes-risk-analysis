# 🩺 Clinical Diabetes Risk Analysis

> **A clinical research project investigating predictors of Type 2 Diabetes using exploratory data analysis and multivariable logistic regression in Jamovi.**

---

## Overview

Type 2 Diabetes is a major chronic disease affecting millions worldwide. Identifying clinical factors associated with diabetes can support earlier detection and improve patient management.

This project uses the **Pima Indians Diabetes Dataset** to demonstrate a complete clinical research workflow, from data quality assessment to predictive modelling. All statistical analyses were performed in **Jamovi**, with results interpreted from a clinical research perspective.

---

## Research Question

**Which demographic and clinical variables are independently associated with Type 2 Diabetes?**

---

## Objectives

- Assess data quality and identify impossible values
- Perform exploratory data analysis (EDA)
- Summarize clinical variables using descriptive statistics
- Investigate relationships using Pearson correlation
- Develop a multivariable logistic regression model
- Evaluate model performance using ROC analysis
- Interpret findings in a clinical context

---

## Dataset

**Source:** Pima Indians Diabetes Dataset

**Sample Size:** 768 participants

### Variables

| Variable | Description |
|----------|-------------|
| Pregnancies | Number of pregnancies |
| Glucose | Plasma glucose concentration |
| BloodPressure | Diastolic blood pressure (mm Hg) |
| SkinThickness | Triceps skin fold thickness (mm) |
| Insulin | 2-Hour serum insulin |
| BMI | Body Mass Index |
| DiabetesPedigreeFunction | Diabetes pedigree function |
| Age | Age (years) |
| Outcome | Diabetes status (0 = No, 1 = Yes) |
